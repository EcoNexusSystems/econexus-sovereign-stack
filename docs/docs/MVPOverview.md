# MVP Overview – One World Lingo

One World Lingo is a multilingual AI transcription and translation tool built for:

- 🇪🇺 GDPR-compliant environments  
- 🌍 Global humanitarian, education, and off-grid deployment  
- 🧠 AI model localization and public sector accessibility

### Features
- Upload + AI transcribe in multiple languages  
- Translate outputs instantly  
- Export .txt for downstream use  
- Fully privacy-compliant, no cloud vendor lock-in

Project is open to DAOs, civic tech orgs, and NGOs.
